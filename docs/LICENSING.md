# Licensing Decision

This scaffold intentionally does not choose a license for VMem. The absence of a license does not grant permission to copy, modify, or redistribute the repository.

Before public release:

1. Select the project license and add the canonical `LICENSE` file.
2. Add SPDX identifiers to package metadata where appropriate.
3. Review third-party dependency licenses and notices.
4. Decide whether a CLA, DCO, or another contribution mechanism is required.
5. Update VXrepo package recipes with the selected license.

Legal review is recommended before accepting external contributions or publishing binaries.
